What is this point of this if the original clone (on webhostapp) is working?

The main reason is preservation. If the original goes down, nobody will be able to play it anymore. This has already happened in the past with the "dodge" test game shown in this video.

The lesser reason is that the original is hosted on 000webhostapp.com. They (the owners of webhostapp) seem to serve pages slower than GitHub's servers do, and they also seem to place a mandatory banner on each page. Overall, I believe that a GitHub pages site would provide a better experience.
